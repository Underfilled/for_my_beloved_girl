**Анонимные голосовые исповеди в Telegram.**

Запиши свою историю, секрет или мысль — кто-то услышит. Слушай чужие шёпоты, реагируй, шепчи в ответ. Полная анонимность.

## Возможности

### 🎤 Записать шёпот
Выбери категорию → запиши голосовое или видео-кружочек → твой шёпот услышат другие!

### 👂 Слушать шёпоты
Получай случайные шёпоты от незнакомцев. Фильтруй по категориям.

### 💬 Реакции и ответы
- Реагируй на шёпоты: ❤️ 🤗 😢 😮
- Шепни в ответ — автор получит анонимно!

### 📬 Входящие
Смотри реакции на свои шёпоты и слушай ответные шёпоты.

## Установка

### Требования
- Python 3.11+
- Токен бота от [@BotFather](https://t.me/BotFather)

### Запуск

```bash
# Клонируй репозиторий
git clone https://github.com/Underfilled/for_my_beloved_girl.git
cd for_my_beloved_girl

# Установи зависимости
pip install -r requirements.txt

# Создай .env файл
cp .env.example .env
# Вставь токен бота в .env

# Запусти бота
python -m bot.main
```

### Docker

```bash
docker build -t shepot .
docker run -d --name shepot -e BOT_TOKEN=your_token shepot
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/help` | Помощь |
| `/stats` | Твоя статистика |

## Конфигурация

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `BOT_TOKEN` | Токен Telegram бота | — |
| `DB_PATH` | Путь к базе данных SQLite | `data/shepot.db` |
| `REPORT_THRESHOLD` | Жалоб до автобана | `5` |
| `MAX_WHISPER_AGE_DAYS` | Макс. возраст шёпота (дни) | `30` |

## Технологии

- [aiogram 3](https://docs.aiogram.dev/) — асинхронный фреймворк для Telegram Bot API
- [aiosqlite](https://aiosqlite.omnilib.dev/) — асинхронная работа с SQLite
- Python 3.12 + asyncio
