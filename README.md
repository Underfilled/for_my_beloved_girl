**Анонимный мессенджер на видео-кружочках внутри Telegram.**

Запиши кружок — получи кружок от незнакомца. Полная анонимность. Никаких текстов, только видео-кружочки.

## Режимы

### 🎲 Рулетка
Запиши кружок и отправь боту — получишь кружок от случайного человека. Твой кружок полетит уже третьему. Каждый раз новые люди!

### 💬 Диалог
Найди собеседника и обменивайтесь кружочками сколько хотите. Нажми "Следующий" чтобы найти нового. Хочешь поделиться контактом? Напиши его на листочке в кружке!

## Возможности

- 🔒 Полная анонимность — бот не раскрывает username
- 🎲 Режим рулетки — каждый раз новые люди
- 💬 Режим диалога — общайся с одним человеком
- ⏭️ Кнопка "Следующий" — смени собеседника в диалоге
- 🚨 Система жалоб — автобан за нарушения
- 📊 Статистика — сколько кружков отправил/получил

## Установка

### Требования
- Python 3.11+
- Токен бота от [@BotFather](https://t.me/BotFather)

### Запуск

```bash
# Клонируй репозиторий
git clone https://github.com/Underfilled/for_my_beloved_girl.git
cd for_my_beloved_girl

# Установи зависимости
pip install -r requirements.txt

# Создай .env файл
cp .env.example .env
# Вставь токен бота в .env

# Запусти бота
python -m bot.main
```

### Docker

```bash
# Собери образ
docker build -t krugovorot .

# Запусти контейнер
docker run -d --name krugovorot -e BOT_TOKEN=your_token krugovorot
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/help` | Помощь |
| `/stats` | Твоя статистика |
| `/cancel` | Выйти из режима |

## Конфигурация

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `BOT_TOKEN` | Токен Telegram бота | — |
| `DB_PATH` | Путь к базе данных SQLite | `data/krugovorot.db` |
| `REPORT_THRESHOLD` | Жалоб до автобана | `5` |
| `DIALOG_TIMEOUT_HOURS` | Таймаут неактивности (часы) | `24` |

## Технологии

- [aiogram 3](https://docs.aiogram.dev/) — асинхронный фреймворк для Telegram Bot API
- [aiosqlite](https://aiosqlite.omnilib.dev/) — асинхронная работа с SQLite
- Python 3.12 + asyncio
